Modal Verbs

â† Back to Grammar Rules

Can / Could

Can: Ability, permission, possibility (present)

Could: Past ability, polite requests, possibility

Example: "I can swim." / "Could you help me?"

May / Might

May: Permission, possibility (more likely)

Might: Possibility (less likely), polite suggestions

Example: "May I go?" / "It might rain."

Must / Have to

Must: Strong obligation, necessity, deduction

Have to: External obligation, necessity

Example: "You must be tired." / "I have to work."

Should / Would

Should: Advice, recommendation, expectation

Would: Polite requests, conditional, past habits

Example: "You should study." / "Would you like some tea?"
